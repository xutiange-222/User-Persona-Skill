想要的"经典"是什么?
梁祝、四季、月光奏鸣曲这种经过时间筛选的曲目;周华健、周传雄的有年代感的老歌;还有中国民族音乐(梁祝、二泉映月、黄河大合唱)——"我们国家也有很多优美的音乐"。
为什么拒绝神曲?
"专区要保持自己的特点"(3/5 共识)。神曲是快消歌,过几周就没人听;Hi-res 用户为高音质付费,期待的是反复回味的高品质内容,不是"什么火什么进来"。
改进建议
精选机制:对流行音乐做品质筛选(品质好、传唱度高、不快消),分批进 Hi-res。不要全集:同一歌手不需要全部专辑,精选 1-2 张代表作即可。
