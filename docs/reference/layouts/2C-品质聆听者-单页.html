核心需求
音乐不是被听,而是来陪伴生活的安静时刻
关键词:放松、助眠、沉思、专注。5/5 共识地把 Hi-res 当作营造氛围的工具,而非"鉴赏对象"——所以追求的不是细节,而是"够通透不糊"。
听歌时刻
临睡前 1 小时是 5/5 共识场景
通勤、开车、健身房是个人差异化场景。工作时间不听 Hi-res——古典曲长太完整不能被打断,改换流行/古风作背景。
内容偏好
主轴是古典,明确拒绝抖音神曲
梁祝、四季、月光、小提琴/钢琴是主轴;经典流行、欧美、民族是副轴。"专区要保持自己的特点,不要为迎合大众变得普通"(3/5)。
设备生态
家用音响、出门用耳机,会验证全链路
华为粉用全套华为(SoundX / FreeBuds);跨品牌玩家选索尼降噪、博士、雅马哈。主动验证"硬件是否支持 Hi-res 全链路"是这群人的共性。
核心痛点
试听入口太隐蔽是最大共识(5/5)
试听入口太隐蔽 5/5(都不知道有免费试听)。其他高频:不能下载(出差/飞机场景)、无 PC 端(工作时只能用网易/酷狗)、介绍偏作者荣誉而非曲子故事(中文译名缺失)、资源同步索尼慢、流行曲库有限。
